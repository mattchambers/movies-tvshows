Matthew's Movies and TV Shows

Requirements
- Python 2.7 

Installation 
1. Ensure you’re in the root of the project folder
2. run the following command in your terminal or cmdPrompt: python setup.py
3. in your browser, enter http://localost:8080/template/index.html
4. When finished with project, hit Ctrl+C in terminal to stop HTTP server on port 8080

Usage 
1. To see the list of Movies, click on the ‘Movies’ button
2. To see the list of Tv Shows, click on the ’Tv Shows’ button
